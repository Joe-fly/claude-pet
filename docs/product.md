# Claude Pet 产品说明文档 (v1.0)

## 一、产品概述

Claude Pet 是基于 Claude Code Logo（大螃蟹）形象打造的桌面宠物应用，定位为用户的 AI 陪伴型桌面助手。

### 核心特性
- **像素螃蟹形象** - 5种动画状态（待机/思考/说话/开心/难过）
- **对话交互** - 气泡回复机制
- **桌面集成** - 窗口置顶、透明度调整、系统托盘

## 二、技术架构

### 2.1 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 桌面封装 | Electron | 跨平台、置顶/透明/托盘 |
| 前端渲染 | Phaser.js | 像素游戏引擎，动画效果 |
| 后端服务 | Flask (内置) | 轻量 API 服务 |
| AI 接入 | Claude CLI (-p 模式) | 调用本地 Claude Code |

### 2.2 项目结构

```
ClaudePet/
├── electron/
│   ├── main.js        # 主进程：窗口、托盘、IPC
│   └── preload.js     # 预加载：安全暴露 API
├── frontend/
│   ├── index.html     # 主页面
│   ├── game.js       # Phaser 游戏逻辑、动画状态机
│   └── styles.css    # 样式
├── backend/
│   ├── app.py         # Flask API 服务
│   ├── claude.py     # Claude CLI 封装
│   └── requirements.txt
└── package.json       # 项目依赖
```

## 四、能力对比分析

### Claude Pet (当前) vs Claude Code CLI

| 特性 | Claude Pet (v1.0) | Claude Code CLI |
|------|-------------------|-----------------|
| **调用方式** | `claude -p` (一次性) | 交互式会话 |
| **会话管理** | ❌ 无 | ✅ 自动保存 |
| **对话历史** | ❌ 每次独立 | ✅ 完整历史 |
| **项目上下文** | ❌ 无 | ✅ 自动理解 |
| **工具能力** | ❌ 无 | ✅ 完整工具集 |
| **MCP 支持** | ❌ 无 | ✅ 支持 |
| **权限系统** | ❌ 无 | ✅ 确认机制 |
| **模式切换** | ❌ 无 | ✅ Default/Auto/Plan |

### 详细说明

#### 对话与上下文
```
Claude Pet:
├── 每次发送都是全新对话
├── 无历史记录
└── 只能回答单轮问题

Claude Code:
├── 自动保存会话 (~/.claude/)
├── 支持会话恢复 (--resume, -c)
├── 自动分析项目结构
└── 多轮对话上下文
```

#### 工具能力
```
Claude Pet (当前):
└── 纯文本问答

Claude Code 完整工具集:
├── Bash - 执行终端命令
├── Edit - 编辑文件
├── Read - 读取文件
├── Write - 创建文件
├── Glob - 文件搜索
├── Grep - 代码搜索
├── Task - 任务委托
├── WebFetch - 网页抓取
└── MCP 工具扩展
```

## 五、已知问题

1. **菜单栏托盘图标** - macOS 菜单栏图标显示不稳定
